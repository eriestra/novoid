Reactive JavaScript

no∅.js — React in Vanilla JS

Every React pattern, natively. No virtual DOM, no JSX, no transpiler.

⚡

signal()

≈ useState

Fine-grained reactive state.

const [count, setCount] = Novoid.signal(0);
🔄

computed()

≈ useMemo

Auto-updating derived values.

const doubled = Novoid.computed(() => count() * 2);
🎯

effect()

≈ useEffect

Side effects with cleanup.

Novoid.effect(() => { /* runs reactively */ });
🏪

createStore()

≈ Redux

Global state with actions.

Novoid.createStore(state, { actions });
🚦

createRouter()

≈ React Router

Routing with params & guards.

Novoid.createRouter(routes, container);
🌀

portal() + errorBoundary()

≈ Portal + Boundary

Render elsewhere, catch errors.

Novoid.portal('#target', content);
⏳

suspense() + lazy()

≈ Suspense

Async with fallback UI.

Novoid.suspense(asyncFn, fallback);
📝

createForm()

≈ Hook Form

Validation + binding.

Novoid.createForm({ email: { required: true } });
📡

useAsync() + bus + toast

≈ Query + Events

Async data, events, notifications.

Novoid.useAsync(() => fetch('/api'));
