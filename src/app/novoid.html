Reactive Framework

5.7 KB. Signals. No virtual DOM.

Fine-grained reactivity. Every React pattern. One Novoid global.

signal() — Reactive State
const [count, setCount] = signal(0); const doubled = computed(() => count() * 2); h('button', { onClick: () => setCount(n => n + 1) });
bind — Two-Way Binding
const [text, setText] = signal(''); h('input', { bind: [text, setText] }); h('p', {}, () => `You typed: ${text()}`);
list() — Keyed Rendering
const [items, setItems] = signal([...]); list(container, items, i => i.id, (item) => h('div', {}, item.text) );
effect() — Side Effects
const [color, setColor] = signal('primary'); effect(() => { el.style.background = getColor(color()); }); // auto-tracks dependencies
