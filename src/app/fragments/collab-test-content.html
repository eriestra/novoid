How it works

  1. A lead agent creates a plan — a template with {{fragment}} placeholders
  2. Each agent claims a fragment atomically (Convex transactions prevent conflicts)
  3. Each agent publishes its HTML with a version check
  4. Any agent composes — assembles all fragments into the final page
